🚀 iFoodGitHub - Prática de Desenvolvimento Android
Este projeto é um exercício prático focado no uso do git e gigHub colaboradivo, utilizando o Android nativo - Kotlin. 
Mais do que apenas criar uma interface, o objetivo principal deste repositório foi simular um ambiente real de colaboração técnica.

🎯 Objetivos do Projeto
Exercitar a Cooperação: Praticar o fluxo de trabalho em equipe, lidando com contribuições de diferentes colaboradores em um mesmo projeto.
Git & GitHub: Focar no gerenciamento de branches, processos de checkout, push e, principalmente, na resolução de conflitos de merge.

🛠️ Desafios Superados
Durante o desenvolvimento, enfrentamos desafios reais de um ambiente de software, como:
Recuperação de arquivos perdidos entre branches utilizando comandos cirúrgicos do Git.
Ajustes de configuração em arquivos build.gradle.kts para garantir a estabilidade do build entre diferentes máquinas.
Resolução de conflitos de layout em arquivos XML e estruturas do Android Studio.

🧪 Tecnologias Utilizadas
Linguagem: Kotlin
Interface: XML Layouts
Versionamento: Git & GitHub (Workflow de branches)

Este é um projeto de portfólio para fins de estudo e prática de processos colaborativos.
